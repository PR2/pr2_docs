PR2 Administration

This set of pages is intended to explain how to serve the role of administrator on the PR2. It assumes basic familiarity with Ubuntu, which is the foundation of the operating system running on the PR2. It also assumes that you have a user on the PR2 in the "admin" group. The remaining sections should provide you information on accessing the PR2, connecting the PR2 to your network, creating new user accounts, managing installed software, and updating the installation image.
